Claude WPM (WordPress Manager) is a powerful tool that integrates with Claude Code to help you manage WordPress sites efficiently.

## What is Claude WPM?

Claude WPM provides a set of commands and utilities that allow Claude Code to:

- Connect to and manage WordPress sites
- Install and configure plugins
- Run WP-CLI commands
- Automate common WordPress tasks

## Key Features

- **Multi-site support** - Manage multiple WordPress installations from a single interface
- **Plugin management** - Install, update, activate, and deactivate plugins
- **WP-CLI integration** - Execute any WP-CLI command through Claude
- **Safe operations** - Built-in safeguards to prevent destructive actions
