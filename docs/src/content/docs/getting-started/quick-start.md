## Basic Usage

Once installed, you can use Claude WPM commands in Claude Code:

```bash
/wpm sites        # List configured sites
/wpm plugins      # Manage plugins
/wpm status       # Check site status
```

## Adding a Site

To add a new WordPress site to manage:

1. Create a site configuration in your WPM settings
2. Test the connection
3. Start managing your site with Claude

## Common Commands

| Command | Description |
|---------|-------------|
| `/wpm sites` | List all configured sites |
| `/wpm plugins list` | List plugins on a site |
| `/wpm status` | Check site health |
