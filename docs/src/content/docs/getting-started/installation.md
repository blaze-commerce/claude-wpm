## Prerequisites

Before installing Claude WPM, ensure you have:

- [Claude Code](https://claude.com/claude-code) installed
- Access to your WordPress sites via SSH or local environment
- WP-CLI installed on your server/local environment

## Installation

Clone the repository to your Claude configuration directory:

```bash
git clone https://github.com/blaze-commerce/claude-wpm.git ~/.claude/commands/wpm
```

## Configuration

After installation, you'll need to configure your WordPress sites. See the [Quick Start](/claude-wpm/getting-started/quick-start/) guide for configuration details.
